INSTRUCTION FOR THE GAME

1. These boxes each represents one word. Your entered guesses will show up here if its correct. If it is incorrect, it wont show up here.


2. There are clues for every word which are to be guessed. Read the clues and try to guess the word.


3. There are 8 chances for guessing the right word. Each wrong guesses entered, will reduce the remaining chances and the game is over once it reaches 0.


4. You can restart the game using this button. You can restart as many times as you want.
